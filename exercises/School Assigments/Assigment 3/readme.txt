Exercises description:
ex 1
Modify all programs so that they display odd numbers in the range from 100 to 1 in descending order.
ex 2
Write a program that asks for the number of numbers to read, then reads those numbers and calculates their sum and average.

ex 3
Write a program that calculates the greatest common divisor (GCD) of two natural numbers a and b. Use both the Euclidean algorithm and the unoptimized Euclidean algorithm.

ex 4
Write a program that outputs the first N odd numbers divisible by 5. The value N should be specified by the user.
